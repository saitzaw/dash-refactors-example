# data-viz
data visualization with flask and dash

# requirement 

- python 3.8.3
- dash 
- plotly 

# workflow 

It reads data from 

- SQL (MySQL, PostgreSQl)
- NoSQL (mongoDB)
- Excel 
- CSV 
- txt 
- Json 
- yaml 

and visualize the data in tables and charts. 

# Feature 

- predict the data (using Linear Regression Method)
- correlation 
- data query 
- data viz
